# Contributing

Thanks for wanting to help make this skill better.

## Pull requests are not open yet

The rule-set is still settling, and the checks need to stay consistent and opinionated while that happens. So outside pull requests are not being accepted right now. This keeps the skill coherent while it finds its shape. That will change once it is stable, and this file will say so when it does.

## What helps right now

- **Open an issue** if a rule here is wrong, out of date, or produced a bad result with your agent. A concrete case (what you asked, what the skill did, what you expected) is the most useful thing you can send.
- **Suggest a check** that belongs in the gates but is missing. Explain the failure it prevents.
- **Share how you use it.** If it caught a real problem, that is worth knowing.

Open an issue for any of these. Feedback shapes what this becomes.
